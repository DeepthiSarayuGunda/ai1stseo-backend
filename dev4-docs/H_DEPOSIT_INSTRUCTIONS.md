# Task H: $5 Deposit for Social Media Posting — Manual Instructions

## THIS IS A MANUAL HUMAN TASK

No automated payment was attempted. This document explains what needs to be done.

## Which Platform Likely Needs the Deposit

**Buffer** (most likely) — if using Buffer as the publishing bridge:
- Buffer free plan: 3 channels, 10 posts/queue — NO payment needed for basic use
- Buffer Essentials plan: $6/month per channel — needed for analytics and more queue slots
- If the $5 is for Buffer, it may be for upgrading one channel

**Twitter/X Developer** (possible):
- X Basic API: $100/month (probably not this)
- X Free tier: $0 but limited to 1,500 tweets/month

**Meta (Facebook/Instagram)** (unlikely):
- Developer accounts are free
- No deposit needed for API access

**Postiz Cloud** (possible):
- Standard plan: $29/month
- 7-day free trial available first

## Most Likely Scenario

The $5 deposit is probably for **Buffer** to test paid features, or for a **Postiz Cloud** trial extension. Buffer is the cheapest option that provides immediate publishing capability.

## Exact Manual Steps

### If for Buffer:
1. Go to [buffer.com](https://buffer.com)
2. Sign up with the team/client email
3. Go to Settings > Billing
4. Add payment method (credit/debit card)
5. Select Essentials plan ($6/month per channel)
6. Connect social accounts (Instagram, Twitter/X)
7. Generate API key at Settings > API

### If for Postiz Cloud:
1. Go to [platform.postiz.com](https://platform.postiz.com)
2. Sign up
3. Start 7-day free trial (no payment needed initially)
4. After trial: Standard plan is $29/month
5. Connect social accounts
6. Get API key from Settings > Developers

## Risk Before Adding Money

- Do NOT pay for any service until social accounts are connected and tested
- Start with free tiers first (Buffer free: 3 channels, Postiz: 7-day trial)
- Only upgrade after confirming the API works with a dry-run test
- Keep receipts for client billing

Tabasum or the client/project owner — whoever controls the payment method for the project.
